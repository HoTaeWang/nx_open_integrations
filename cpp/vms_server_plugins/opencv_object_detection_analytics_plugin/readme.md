# Analytics plugin for Nx VMS Server (opencv\_object\_detection\_analytics\_plugin)

## Features
- Detect objects (cats, dogs, people).
- Send object detection events.
- Send plugin diagnostics events.

## Supported platforms
Ubuntu 18.04, g++-7 compiler.
